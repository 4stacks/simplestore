simpleStore

A clean, responsive storefront boilerplate with no database or backend

Download .zip Online Demo View on GitHub

A free and open-source simple online store build on HTML, CSS and Javascript. No database. No backend.

simpleStore is built on simpleCart.js and Skeleton CSS Framework for a lightweight, fast, simple to use, and completely customizable experience. For more information/documentation, visit their websites.
